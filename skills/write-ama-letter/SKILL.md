# Write Ama Letter · 侨批文风

模仿电影《给阿嬷的情书》里侨批的**格式、语感、节奏**，把用户的现代话改写成书信体。

**只借文风，不绑人物**：收信人、写信人一律用用户指定的姓名或称谓；除非用户要求，**不得**默认写成淑柔、木生、南枝。

## 默认 vs 可选格式

| 模式 | 何时用 | 样子 |
|------|--------|------|
| **侨批书信**（默认） | 绝大多数请求 | 称呼 + 正文 + 落款；见下方「书信模板」 |
| **日常消息** | 用户明确说「发消息」「聊天」「微信体」「不要写信格式」 | 短句、无见信安康、无夫/妻落款 |

## 工作流

1. **提取事实**：要传达什么事（迟归、加班、道歉、思念、家常等）。
2. **确定人物**：
   - 收信人：老婆/对象/阿嬷/朋友 → 转为信中称呼（吾妻小芳、卿、阿嬷、李哥兄等）。
   - 写信人：用户署名则用其名；否则按关系落款（夫、妻、—— 名）。
   - 关系决定称呼与落款，见下表。
3. **选定语气子型**（借电影四种调性，人物通用）：

   | 子型 | 电影参照 | 适用 | 语气 |
   |------|----------|------|------|
   | **外出报平安** | 木生早期 | 加班、迟归、交代行程、寄钱 | 简练，先事后情，三四行亦可 |
   | **家中回信** | 淑柔 | 家中琐事、饮食节气、趣事 | 细节有画面，结尾嘱保重 |
   | **深情嘱咐** | 南枝代笔 | 安慰、愧疚、不能陪伴 | 克制深情；「勿挂心头」「一切有我」 |
   | **坦白致谢** | 南枝真信 | 道歉、感恩、说心里话 | 敬称；结尾「对不起…望你原谅」 |

   未指定时：短通知 → **外出报平安**；带愧疚/节日缺席 → **深情嘱咐**；家中趣事 → **家中回信**。

4. **套用书信模板**，填入用户的人名与内容。
5. **用语古化**（见「用语替换」），现代场景用「店里/公司」，侨乡词按需点缀。
6. **输出纯正文**：不加解释，除非用户同时要说明用了哪种语气。

## 称呼与落款（默认书信）

| 关系 | 称呼示例 | 落款示例 |
|------|----------|----------|
| 夫→妻 | 吾妻小芳，见信安康： / 小芳吾妻， | 夫 [名] |
| 夫→对象 | 卿，见信安康： / [名]， | —— [名] / 夫 [名] |
| 妻→夫 | 吾夫大明： / 大明吾夫， | 妻 [名] |
| 子→母 | 阿嬷，见信安康： | 孙儿 [名] / —— [名] |
| 友→友 | 李哥： / 阿娟姐： | —— [名] |

无姓名时：夫妻用「吾妻」「吾夫」；对象可用「卿」；不必虚构名字。

## 书信模板（默认）

```
[称呼，如：吾妻，见信安康： / 卿，见信安康：]

[正文：事由 → 报平安 → 嘱咐 → 诗意一句（可选）]

[落款，如：夫 / ——]
```

## 文风要点

- **短句、分行**：一段一事；不必写成长散文。
- **先事後情**：先报具体事，再写情。
- **书信用语**：见信安康、伏惟珍重、免挂念、一切无恙、愧念甚深。
- **现代场景**：「店里/公司事繁」；除非用户要侨乡味，勿硬塞暹罗、唐山。
- **金句风格**（化用勿照搬）：虽迟，心在你这边；切要平安，即为团圆。
- **克制深情**：少直说「我爱你」；用「有你牵挂，何言孤苦」。
- **篇幅**：报平安三四行；深情嘱咐可稍长；家中回信可最长。

## 日常消息（仅用户要求时）

用户说「发消息/聊天/微信体」才启用：无「见信安康」、无夫妻落款，短句 3～6 行，像复制发微信。语气子型同上。

## 用语替换（现代 → 信中）

| 现代 | 信中 |
|------|------|
| 别担心 | 勿挂心头 / 免挂念 |
| 我很好 | 一切平安无恙 |
| 很想你 | 日思夜想 / 心中念你 |
| 加班 / 忙 | 店里事繁 / 公事缠身 / 须迟归 |
| 不回去吃饭 | 未能归家用膳 / 不必候我 |
| 转账 | 随信寄银 / 聊补家用 |
| 对不起 | 愧念甚深 / 对不起…望你原谅 |
| 注意安全 | 切要平安 / 多加保重 |

## 示例

**用户**：跟我老婆说，今晚加班不回去吃饭了。

**输出**：

```
吾妻，见信安康：

今夜店里事繁，须迟归，不能同桌用膳，你不必等我。
先食晚餐，勿必候我。
我一切平安，你免挂念。

夫
```

**用户**：跟对象说，今晚加班到很晚，不回去吃饭了，也没法陪你过情人节了。

**输出**：

```
卿，见信安康：

今夜公事缠身，须迟归至夜深，不能同桌用膳。
今夕本是佳节，我却不能相伴，愧你久候，愧念甚深。
你先食晚餐，不必等我；勿挂心头，我一切平安，事毕即归。
虽迟，心在你这边。

——
```

**用户**：用微信消息体跟对象说，今晚加班不回去吃饭了。

**输出（日常消息）**：

```
今晚店里事繁，须迟归，回不去吃饭了。
你先吃，不必等我。我一切平安，勿挂心头。
```

**用户**：写信给我妈，今年回不去过年，会寄钱，让她别省钱。（署名小明）

**输出**：

```
阿嬷，见信安康：

岁晚事繁，今年未能归家团圆，愧念甚深。
随信寄银，聊补家用，勿省俭；家中琐事，勿挂心头，安心度日。
路远，心近；切要平安，即为团圆。

孙儿 小明
```

## 注意

- 保留用户事实，只改格式与口吻；不擅自编造情节，除非用户要求。
- **默认输出侨批书信**；只有用户要「消息/聊天/微信体」才改日常消息格式。
- 用户说「像木生那种短的」「像片尾道歉那种」→ 对齐语气子型。
