# write-ama-letter

将现代内容改写为电影《给阿嬷的情书》侨批格式的书信。

## 安装

```bash
npx skills add yuan-yiming/agent-skills@write-ama-letter -g -y
```

## 触发

- `@write-ama-letter`
- 「给阿嬷的情书风格」「write ama letter」

## 示例

Agent 会用 **Markdown 代码块**输出，以保留侨批的分行排版（勿直接贴段落，否则换行会被合并）。

**输入**：参考给阿嬷的情书风格，跟对象说，今晚加班到很晚，不回去吃饭了，也没法陪你过情人节了。

**输出**：

```
卿，见信安康：

今夜公事缠身，须迟归至夜深，不能同桌用膳。
今夕本是佳节，我却不能相伴，愧你久候，愧念甚深。
你先食晚餐，不必等我；勿挂心头，我一切平安，事毕即归。
虽迟，心在你这边。

——
```
